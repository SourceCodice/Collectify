# Sicurezza

Collectify e' in fase iniziale di sviluppo.

## Segnalare vulnerabilita

Per segnalare un problema di sicurezza, apri una issue privata o contatta il maintainer del repository GitHub.

Non pubblicare dettagli sfruttabili in issue pubbliche prima che il problema sia stato verificato.

## Dipendenze

Il progetto usa:

- .NET 8 per il backend.
- Electron, React, TypeScript e Vite per il frontend.

Le dipendenze npm e GitHub Actions sono monitorabili tramite Dependabot.
